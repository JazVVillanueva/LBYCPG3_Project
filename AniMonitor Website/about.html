About AniMonitor

A real-time environmental monitoring web application powered by an ESP32 sensor node. AniMonitor tracks UV radiation, temperature, and humidity — designed to keep communities informed and safe.

☀️ UV Index 🌡️ Temperature 💧 Humidity 📡 ESP32 🔥 Firebase
🎯
Project Goal
Real-Time Env. Monitoring
Live sensor data from ESP32 nodes, accessible anywhere via the web.
📡
Hardware
ESP32 + BME280 + GUVA-S12SD
Temp, humidity, and UV sensors polled every 2 seconds.
